# Smartin marketplace

Free Claude Cowork / Claude Code plugins by Smartin.

## Install

```
/plugin marketplace add martinsoucekhbhuzg/smartin-marketplace
/plugin install smartin-compass
```

After install, reload plugins if prompted (`/reload-plugins`).

## Plugins

- **smartin-compass** — Catch raw ideas as they come, see which ones are worth
  checking, get one clean weekly Finding worth forwarding. Local-first, free.

More at https://smartin.work
